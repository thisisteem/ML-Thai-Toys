Thai Toys Classification

This project is for Machine Learning made by Chakphet Phachanawan

This model will predict the image of Thai Toys by using webcam and then the result can be 5 classes which is

  1. งูไม้ระกำ
  2. ม้าก้านกล้วย
  3. ลิ้นมังกร
  4. ว่าว
  5. หนังสติ๊ก
